# 🧪 視覚的陰イオン・陽イオン分解シミュレーター

HTML5 Canvas と TypeScript（Vite）を用いて構築された、溶液中におけるイオンの熱運動（ブラウン運動）および電気分解時の電移動を視覚的に学べるシミュレーターです。

---

## 🚀 特徴

- **2つのモード**:
  - **電源 OFF**: 溶液中でイオンが不規則にゆらゆらと動く「熱運動（ブラウン運動）」を模擬。
  - **電源 ON**: 電場が発生し、陽イオン（$Na^+$）が陰極へ、陰イオン（$Cl^-$）が陽極へ引き寄せられる「電気分解」を模擬。
- **物理挙動の簡易再現**: 粒子の速度制限（摩擦抵抗）や壁との反発、電場による加速度を簡易的な物理演算で処理しています。
- **TypeScript による堅牢な設計**: クラスベースで粒子（Ion）とシミュレーター本体（Simulator）を疎結合に管理しています。
